NStudio is a free (open source) editor that was written in C# on the .NET 2.0 Framework.

Main features:
- 5 editing components (Scintilla editor, ICSharpCode editor, Rich text editor, Standard Windows editor and WYSIWYG HTML Editor)
- Syntax Highlighting
- Multi documents dockable to any window edge
- Web Browser
- Very good and beauty user interface (support Office2007 style)
- Highly configurable and customizable.
- Plugins (3 plugins included in instalation: FileSearch, CodeSnippet adn WindowsMediaPlayer)
- Multi-language (English, Bulgarian, Hungarian, Latvian, Russian)
- Drag and Drop supported
- Brace and Indent guideline Highlighting
- Clipboard ring, Math tools, System Information tool, Directory Watcher and Console

Requirements:
- .NET Framework 2.0 or higher
- IE 5 or higher
- Windows Media Player 9 or higher for WindowsMediaPlayer plugin
